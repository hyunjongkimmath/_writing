\begin{definition}[Trivial absolute value] \label{definition:trivial_absolute_value_on_a_field}
Let $K$ be a field. The \hldef{trivial absolute value on $K$} is the \hyperrefIfExists{definition:absolute_value_on_a_field}{absolute value}\CrefIfExists{definition:absolute_value_on_a_field}
$$\hlin{|\cdot|_{\mathrm{triv}} : K \to \mathbb{R}_{\geq 0}}$$
defined by
\[
|x|_{\mathrm{triv}} \;:=\; \begin{cases}
0 & \text{if } x = 0, \\
1 & \text{if } x \neq 0.
\end{cases}
\]
\end{definition}
