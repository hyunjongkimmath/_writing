\begin{definition} \label{definition:regular_system_of_parameters_of_a_noetherian_local_ring}
Let $(A, \mathfrak{m})$ be a \CrefAndHyperrefIfExist{definition:noetherian_ring}{Noetherian} \CrefAndHyperrefIfExist{definition:local_ring}{local ring} of \CrefAndHyperrefIfExist{definition:krull_dimension_of_a_commutative_ring}{dimension} $d$. A set of elements $\{x_1, \dots, x_d\} \subset \mathfrak{m}$ is called a \hldef{regular system of parameters} if the images of these elements in $\mathfrak{m}/\mathfrak{m}^2$ form a \CrefAndHyperrefIfExist{definition:basis_of_a_vector_space_over_a_field}{basis} of this $A/\mathfrak{m}$-vector space.
\end{definition}
