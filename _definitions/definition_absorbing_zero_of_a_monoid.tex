\begin{definition} \label{definition:absorbing_zero_of_a_monoid}
Let $\Gamma$ be a \CrefAndHyperrefIfExist{definition:monoid}{monoid}. An element $z \in \Gamma$ is called an \hldef{absorbing zero} (or simply a \hldef{zero element}) if for all $\gamma \in \Gamma$:
$$\hlin{z + \gamma = z \quad \text{and} \quad \gamma + z = z.}$$
Such an element, if it exists, is unique and is typically denoted by \hl{$0$} (using additive notation) or \hl{$\infty$} (in tropical contexts).
\end{definition}
