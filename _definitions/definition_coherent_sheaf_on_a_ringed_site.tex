\begin{definition} \label{definition:coherent_sheaf_on_a_ringed_site}
Let $(\mathcal{C}, \mathcal{O})$ be a \CrefAndHyperrefIfExist{definition:ringed_site}{ringed site}. An \CrefAndHyperrefIfExist{definition:module_over_a_sheaf_of_rings_on_a_site}{$\mathcal{O}$-module sheaf} $\mathcal{F}$ is \hldef{coherent} if it is \CrefAndHyperrefIfExist{definition:quasi_coherent_sheaf_on_a_ringed_site}{quasi-coherent} and one can choose the index sets $I_i, J_i$ to be finite for each $i$. Equivalently, $\mathcal{F}$ is locally finitely presented as an $\mathcal{O}$-module sheaf.

\end{definition}
