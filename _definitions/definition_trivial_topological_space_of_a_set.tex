\begin{definition}[Indiscrete (or trivial) topological space] \label{definition:trivial_topological_space_of_a_set}
Let $X$ be a set. The \hldef{indiscrete topology} (also called the \hldef{trivial topology}) on $X$ is the topology \hl{$\mathcal{T}_{\mathrm{indisc}}$} defined by
\[
\mathcal{T}_{\mathrm{indisc}} \;:=\; \{ \emptyset, X \},
\]
i.e., only the empty set and the entire set are open. The pair $(X, \mathcal{T}_{\mathrm{indisc}})$ is called an \hldef{indiscrete topological space}.
\end{definition}
