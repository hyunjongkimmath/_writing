\begin{definition}[Discrete absolute value] \label{definition:discrete_absolute_value_on_a_field}
Let $K$ be a field equipped with an \hyperrefIfExists{definition:absolute_value_on_a_field}{absolute value}\CrefIfExists{definition:absolute_value_on_a_field}$|\cdot|$. The absolute value $|\cdot|$ is called \hldef{discrete} if its image $|K^\times| = \{ |x| : x \in K^\times \} \subseteq \mathbb{R}_{> 0}$ is a discrete subgroup of the multiplicative group $\mathbb{R}_{>0}$ (with the usual topology).
\end{definition}
