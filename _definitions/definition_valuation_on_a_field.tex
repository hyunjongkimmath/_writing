\begin{definition}[Valuation on a field] \label{definition:valuation_on_a_field}
Let $K$ be a field. A \hldef{valuation on $K$} is a function
\[
v : K \to \Gamma \cup \{\infty\},
\]
where $(\Gamma, +, \leq)$ is a totally ordered abelian group and $\infty$ is an element greater than all elements of $\Gamma$, satisfying for all $x, y \in K$:
\begin{enumerate}
    \item $v(x) = \infty$ if and only if $x = 0$,
    \item $v(xy) = v(x) + v(y)$, \label{part:valuation_on_a_field_turns_multiplication_into_addition}
    \item $v(x + y) \geq \min\{v(x), v(y)\}$. \label{part:vlauation_on_a_field_satisfies_a_strict_triangle_inequality}
\end{enumerate}

Alternatively (and essentially equivalently), a valuation on $K$ is also defined as a function $v: K^\times \to \Gamma$ with properties \eqref{part:valuation_on_a_field_turns_multiplication_into_addition} and \eqref{part:vlauation_on_a_field_satisfies_a_strict_triangle_inequality} and is extended into a function $v: K \to \Gamma \cup \{\infty\}$ by setting $v(0) = \infty$.

The pair $(K, v)$ is called a \hldef{valued field}.
\end{definition}
