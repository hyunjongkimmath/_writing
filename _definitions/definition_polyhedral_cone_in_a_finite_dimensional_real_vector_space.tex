\begin{definition}[Polyhedral Cone] \label{definition:polyhedral_cone_in_a_finite_dimensional_real_vector_space}
Let $V$ be a finite-dimensional real vector space. 
A \hldef{polyhedral cone} $\sigma \subset V$ is a set of the form
\[ \sigma = \left\{ \lambda_1 v_1 + \cdots + \lambda_k v_k \mid \lambda_i \geq 0 \text{ for all } i \right\} \]
for some finite set of vectors $v_1, \ldots, v_k \in V$, i.e. $\sigma$ is the \CrefAndHyperrefIfExist{definition:cone_generated_by_a_subset_of_a_real_vector_space}{cone generated by} $\{v_1, \ldots, v_k\}$. 
\end{definition}
