\begin{definition} \label{definition:saturated_multiplicative_system_in_a_category}
A \CrefAndHyperrefIfExist{definition:multiplicative_system_of_morphisms_in_a_category}{multiplicative system} $S$ in a \CrefAndHyperrefIfExist{definition:category}{category} $\mathscr{C}$ is called \hldef{saturated} if it satisfies the following condition: any morphism $f$ in $\mathscr{C}$ whose image $Q(f)$ in the \CrefAndHyperrefIfExist{definition:localization_of_a_category_by_a_multiplicative_system}{localized category} $\mathscr{C}(S^{-1})$ is an isomorphism belongs to $S$.
\end{definition}
