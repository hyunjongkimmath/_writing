\begin{definition} \label{definition:riemann_zeta_function}
Let $s \in \mathbb{C}$. The \hldef{Riemann zeta function} $\zeta(s)$ is defined as
$$\hlin{\zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s}.}$$
which converges absolute when $\Re(s) > 1$ and in this case, $\zeta(s)$ can be expressed as an Euler product:
$$\zeta(s) = \prod_{n=1}^\infty \frac{1}{(1-p^{-s})}.$$

Here are alternate definitions of $\zeta(s)$:

    \item It is a special case of the \hyperrefIfExists{definition:dedekind_zeta_function_of_a_number_field}{Dedekind zeta function $\zeta_K(s)$} in the case that the number field $K$ is simply the field $\bbQ$ of rational numbers\CrefIfExists{definition:dedekind_zeta_function_of_a_number_field}.
\end{enumerate}
\end{definition}
