
\begin{definition}[Fixed Point Set] \label{definition:fixed_point_set_of_a_group_acting_on_a_set}
Let $(G,\cdot)$ be a group acting on a set $X$. For $H \subseteq G$, the \hldef{fixed point set of $H$ in $X$} is defined by
$$\hlin{X^H = \{ x \in X : h \cdot x = x \text{ for all } h \in H \}}.$$
\end{definition}
