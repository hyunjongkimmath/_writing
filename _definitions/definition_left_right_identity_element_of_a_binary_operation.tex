\begin{definition} \label{definition:left_right_identity_element_of_a_binary_operation}
Let $S$ be a set equipped with a binary operation $\cdot : S \times S \to S$. An element $e \in S$ is called a \hldef{left identity element} if $e \cdot x = x$ for all $x \in S$. Similarly, an element $e \in S$ is called a \hldef{right identity element} if $x \cdot e = x$ for all $x \in S$. An element $e \in S$ is called an \hldef{identity element} (or \hldef{neutral element}) if it is both a left and a right identity element. If an identity element exists, it is unique.
\end{definition}
