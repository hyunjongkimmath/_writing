\begin{definition}[Strongly Convex Cone] \label{definition:strongly_convex_cone_in_a_finite_dimensional_real_vector_space}
Let $V$ be a \CrefAndHyperrefIfExist{definition:basis_of_a_vector_space_over_a_field}{finite-dimensional} real \CrefAndHyperrefIfExist{definition:vector_space_over_a_field}{vector space}. 
A \CrefAndHyperrefIfExist{definition:convex_cone_in_a_finite_dimensional_real_vector_space}{convex cone} $\sigma \subset V$ is called \hldef{strongly convex} if $\sigma \cap (-\sigma) = \{0\}$, i.e. $\sigma$ contains no nontrivial linear subspace.
\end{definition}
