\begin{definition} \label{definition:transitive_closure_of_a_set}
The \hldef{transitive closure of a set $x$} denoted \hl{$\text{tc}(x)$}, is the unique minimal transitive set such that $x \subseteq \text{tc}(x)$. It can be constructed as the union $\bigcup_{n \in \mathbb{N}} \cup^n x$, where $\cup^0 x = x$ and $\cup^{n+1} x = \bigcup (\cup^n x)$.
\end{definition}
